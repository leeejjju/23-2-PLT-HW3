# HOW TO USE



### 1. unzip it

unzip the zip file 



### 2. compile  


compile the cpp file with makefile 
> $ make



### 3. use 

you can use parser and interpretor with this way:


* for parsing


> $ ./my_interpretor -p [expression]


* for interpreting


> $ ./my_interpretor [expression]
